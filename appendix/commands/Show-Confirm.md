# Show-Confirm 
 
Shows a user a confirmation request message box. 
 
## Syntax 
 
Show-Confirm [-Title] &lt;String&gt; 
 
 
## Detailed Description 
 
Shows a user a confirmation request message box.
Returns "yes" or "no" based on user's response.
The buttons that are shown to the user are "OK" and "Cancel".
