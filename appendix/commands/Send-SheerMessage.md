# Send-SheerMessage 
 
Sends a sheer message to the app in which context the script is executing. 
 
## Syntax 
 
Send-SheerMessage [-Name] &lt;String&gt; [-GetResult] 
 
 
## Detailed Description 
Sends a sheer message to the app in which context the script is executing.
