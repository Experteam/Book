# New-Domain 
 
Creates a new domain with the specified name. 
 
## Syntax 
 
New-Domain [-Name] &lt;String&gt; [-LocallyManaged] 
 
 
## Detailed Description 
The New-Domain cmdlet creates a domain if it does not exist.
