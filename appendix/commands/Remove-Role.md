# Remove-Role 
 
Removes a Sitecore role. 
 
## Syntax 
 
Remove-Role [-Identity] &lt;AccountIdentity&gt; 
 
Remove-Role -Instance &lt;Role&gt; 
 
 
## Detailed Description 
 
The Remove-Role command removes a Sitecore role.
