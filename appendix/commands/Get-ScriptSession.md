# Get-ScriptSession 
 
Returns the list of PowerShell Extension Sessions running in background. 
 
## Syntax 
 
Get-ScriptSession -Id &lt;String&gt; 
 
Get-ScriptSession -Current 
 
Get-ScriptSession -Type &lt;String[]&gt; 
 
 
## Detailed Description 
Returns the list of PowerShell Extension Sessions running in background.
