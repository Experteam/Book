# Get-Preset 
 
Returns a serialization preset for use with Serialize-Item. 
 
## Syntax 
 
Get-Preset [[-Name] &lt;String[]&gt;] 
 
 
## Detailed Description 
Returns a serialization preset for use with Serialize-Item.
