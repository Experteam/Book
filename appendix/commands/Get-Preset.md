# Get-Preset 
 
Returns a serialization preset for use with Export-Item. 
 
## Syntax 
 
Get-Preset [[-Name] &lt;String[]&gt;] 
 
 
## Detailed Description 
 
The Get-Preset command returns a serialization preset for use with Export-Item.
