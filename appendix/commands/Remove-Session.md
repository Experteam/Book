# Remove-Session 
 
Removes one or more Sitecore user sessions. 
 
## Syntax 
 
Remove-Session -InstanceId &lt;String[]&gt; 
 
Remove-Session [-Instance] &lt;Session&gt; 
 
 
## Detailed Description 
 
The Remove-Session command removes user sessions in Sitecore.
