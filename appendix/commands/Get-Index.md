# Get-Index 
 
Returns Sitecore database indices. 
 
## Syntax 
 
Get-Index [[-Database] &lt;Database&gt;] [[-Name] &lt;String&gt;] 
 
 
## Detailed Description 
Returns Sitecore indices in context of a particular database.
